Slide Animation

SlideAnimation.js adds a smooth transition animation

Now scroll to experience the animation

There is no dependency on this library

In the simplest case, you just need to call the function and pass it an element

You can also pass an object to configure the animation

TranslateX to change the distance and direction of the horizontal axis

Of course, translateY to change the distance and direction of the vertical axis

Scale to resize the element as it slides

Duration to reset the time to execute the animation

Show affect whether the animation is performed the first time

And you will need onload if the element takes time to load
